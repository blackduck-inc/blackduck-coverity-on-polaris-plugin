    Configuring

    Once the Synopsys Polaris for Jenkins plugin is installed, and before you use it in a job, you must configure the following:

    1. Synopsys Polaris CLI
    2. Synopsys Polaris for Jenkins

    Configuring Synopsys Polaris CLI

    Use the following process to configure Synopsys Polaris CLI:

    1. In Jenkins, navigate to Manage Jenkins > Global Tool Configuration.
    2. Scroll down to the Polaris CLI section, click Add Polaris CLI, and complete the following:
      1. Name: A name for the Synopsys Polaris CLI installation.
      2. Install automatically: To enable Synopsys Polaris for Jenkins to install the Synopsys Polaris CLI from your Synopsys Polaris server automatically, leave this box checked, and proceed to the next step. To point Synopsys Polaris for Jenkins to an existing Synopsys Polaris CLI installation, uncheck this box and enter the path to the Synopsys Polaris CLI directory in the Installation directory field.
    3. Click Save.

    Configuring Synopsys Polaris for Jenkins

    Use the following process to configure Synopsys Polaris for Jenkins:

    1. Navigate to Manage Jenkins > Configure System.
    2. Scroll down to the Synopsys Polaris section and complete the following:
      1. Polaris URL: The URL to your Synopsys Polaris instance.
      2. Polaris credentials: The Jenkins secret text credentials that you created to store your Synopsys Polaris access token.
      3. Advanced: If you want to change the timeout that Synopsys Polaris for Jenkins uses while waiting for a response from the Synopsys Polaris server, click Advanced and enter the timeout in seconds in the Polaris connection timeout field.
      4. Test connection to Polaris: To test that Synopsys Polaris for Jenkins can connect to Synopsys Polaris with the configuration data you have entered, click Test connection to Polaris.
    3. Click Save.
